## 游戏规则

主动消除: 5个即以上的相同颜色的方块连在一起，一个球`2`分，暂停产生新的球
被动消除: 如果因为生成而消除，零分

## Quick start

### 初始化

```bash
  cd template
  npm install
  npm run dev
```

如果使用cnpm:

```bash
  cd template
  cnpm install
  npm run dev
```
